# Исследовательский корпус

Локальная папка содержит предоставленные пользователем материалы:

- `score/Webern_Op10_score.pdf` — партитура Пяти пьес для оркестра op. 10;
- `texts/Reutter_Webern_Op10_full.pdf` — развёрнутая работа Ханса Петера
  Ройтера;
- `texts/Reutter_Webern_Op10_short.pdf` — сокращённая статья Ройтера;
- `texts/Webern_Der_Weg_zur_neuen_Musik.pdf` — лекции Веберна;
- `texts/Kholopova_Kholopov_Anton_Webern.pdf` — В. Холопова и Ю. Холопов,
  «Антон Веберн. Жизнь и творчество».

PDF-файлы исключены из git до выбора приватного/публичного режима и проверки
прав на распространение. Аналитические выводы, указатели и машинные профили
хранятся в `notes/` и могут версионироваться независимо от самих источников.

Для версии 5 добавлены `notes/REGISTER_PHRASE_TEMPO_MODEL_RU.md` и
`notes/PERFORMANCE_GRAMMAR_AND_VOCABULARY_RU.md`: там отдельно
описаны рабочие диапазоны, микро-фразы, условия общего ритма, педаль III,
ремарки, словарь, инструментальные capability-правила и агогика с разделением
наблюдений и эвристических чисел.
